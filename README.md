# MeSNEmu

Requirements For Building IPA:
- ldid <br>
- homebrew <br>
- xcode <br>
- xcpretty <br>
<br>
Building IPA: 
1) Clone project (git clone --recursive 'clone url') <br>
2) Run ./genipa.sh in terminal or the start button in Xcode
